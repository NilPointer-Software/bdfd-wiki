# $customImage
*(for premium bots)*

This function is used to return the uploaded image in the app.

## Syntax
```
$customImage[Custom image tag]
```

## Parameters
- `custom image tag`: The image tag you set in the app while uploading the image.

## Example
```
$nomention
$customImage[NiceImage]
```
