# setEmbedImage
Adds an image to an embed.

## Syntax
```js
setEmbedImage(imageURL)
```

### Parameters
- `imageURL` : The image to set in the embed. Value must be a valid image link. (Optional)

## Example
```js
setEmbedImage('https://http.cat/200');
setEmbedResponse();
```
