# $channelTopic[]
Returns the topic of a channel by its ID.

## Syntax
```
$channelTopic[Channel ID]
```

### Parameters
- `Channel ID` `(Type: Snowflake || Flag: Required)`: The channel to return its topic.

## Example
```
$nomention
<#$noMentionMessage> topic is : $channelTopic[$noMentionMessage]
```
