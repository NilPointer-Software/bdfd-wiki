# $channelID
Returns the ID of the current channel.

## Syntax
```
$channelID
```
## Example
```
$nomention
Channel ID: $channelID
```

```discord yaml
- user_id: 803569638084313098
  username: RainbowKey

  color: "#E67E22"

    !example
- username: BDFD Support

      Channel ID: 566370477967147018
```
