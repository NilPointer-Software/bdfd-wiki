# $userID
Fetches a user's ID using their username or user-tag.

## Syntax
```
$userID[Username#Discriminator / Username]
```

### Parameters
- `Username#Discriminator / Username` : The user's username (User) or user-tag (User#0000) to return the ID for.

## Example
```
$nomention
$userID[$username]
```
