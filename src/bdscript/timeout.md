# $timeout

Timeouts the mentioned users, or the user provided in 'User ID', for a certain 'Duration' (max 28 days)

## Usage
```
$timeout[duration;(userID)]
```

## Example
```
$nomention
$timeout[$message[1];$mentioned[1]]
```
