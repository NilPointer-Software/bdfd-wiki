# $userExists
Checks if a user exists in Discord using it's ID. Returns "true", if it exists otherwise "false".

## Syntax
```
$userExists[User ID]
```

### Parameters
- `User ID` `(Type : String,Snowflake || Flag : Emptiable)` : The ID of the user to check.

## Example
```
$nomention
$userExists[$mentioned[1]]
```
