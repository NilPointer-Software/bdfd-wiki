# $unban
Unbans a user by providing their username *(from the author's message)*.

## Syntax
```
$unban
```

### Permissions
Required permission which the bot must have for this function to work properly.
- `ban`

## Example
```
$nomention
$unban
Successfully unbanned user!
```
