# $eval
Evaluates the provided BDScript code.
> ⚠️ Can only be used in BDScript 2, this function should be used with caution.

## Usage
```
$eval[BDScript source code]
```

### Breakdown 
- `BDScript source code` - The code to be evaluated.

## Example
```
$nomention
$eval[$message]
```
