# $boostCount[]
Returns the guild's number of nitro boosts.

## Syntax
```
$boostCount[guild ID]
```

 ### Parameters
- `guild ID` `(Type : Snowflake || Flag : Required)` : The guild to get the number of boosts for.

## Example
```
$nomention
This server currently has $boostCount[$message[1]] boost(s).
```
