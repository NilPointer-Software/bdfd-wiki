# $variablesCount
Returns how many of a certain variable type the bot has.

## Syntax
```
$variablesCount[Type]
```

### Parameters
- `Type` : The variable type to return the count for. Accepts either `user`, `server`, or `globaluser` as input.

## Example
```
$nomention
$variablesCount[server]
```
