# $variablesCount
Returns how many of a certain variable type the bot has.

## Usage
```
$variablesCount[variableType (user/server/globaluser)]
```

### Breakdown
- `variableType` - The variable type to return the count for.

## Example
```
$nomention
$variablesCount[server]
```
