# $min
Returns the smallest number from the provided numbers.

## Usage
```
$min[A;B;...]
```

### Parameters 
- `A,B,...` `(Type : Integer || Flag : Required)` : The numbers to get the data from. Separate numbers using `;`.

## Example
```
$nomention
$min[3;5;1]
```
