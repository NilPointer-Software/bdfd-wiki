# $ignoreLinks
Ignores image links in the message.

## Syntax
```
$ignoreLinks
```

## Example
```
$nomention
$ignoreLinks
Here's a toaster IRL:
https://media.discordapp.net/attachments/1011682358031826994/1027580044928888832/856506821023629332.png
```
