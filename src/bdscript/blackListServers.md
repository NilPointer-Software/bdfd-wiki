# $blackListServers
Disables this command for the provided servers.

## Usage
```
$blackListServers[serverIDs;errorMessage]
```

### Parameters
- `serverIDs` `(Type : Snowflake || Flag : Emptiable)` : The servers to blacklist. Separate server IDs using `;`.
