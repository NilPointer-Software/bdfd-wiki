# $findChannel
Finds a channel's ID from the given channel name, ID, or mention.

> 📝 This function can only find channels of the current server.

## Syntax
```
$findChannel[Channel]
```

### Parameters
- `Channel` `(Type: String || Flag: Emptiable)`: The channel name/ID/mention to find.

## Example
```
$nomention
$findChannel[$message]
```
