# $categoryCount
Returns the category count of the current guild.

## Syntax
```
$categoryCount
```
## Example
```
$nomention
There are $categoryCount categories in this server!
```
``` discord yaml
- user_id: 729343563401265193
  username: Nicky
  color: "#EE7908"
  content: |
    !categories

- user_id: 566613317972394004
  username: Wiki Bot
  color: "#748BD4"
  bot: true
  verified: true
  content: |
    There are 3 categories in this server!
```
