# $round
Rounds up the provided number.

## Usage
```
$round[number;(decimalPlace)]
```

### Breakdown
- `number` - The number to round.
- `decimalPlace` - The decimal place to round the number to. Default as `0`. Optional.

## Example
```
$nomention
$round[100.123;1]
```
