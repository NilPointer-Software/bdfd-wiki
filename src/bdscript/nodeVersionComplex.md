# $nodeVersion[]
Returns the version of the specified node.

## Syntax
```
$nodeVersion[Node Number]
```

### Parameters
- `Node Number` `(Type: Integer || Flag: Required)`: Which node version will be returned.

## Example
```
$nomention
13 Node Version: `$nodeVersion[13]`
```
