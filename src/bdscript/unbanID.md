# $unbanID
Unbans a user by using their ID taken from the last argument of the author's message.

## Syntax
```
$unbanID
```

### Permissions
Required permissions that the bot must have for this function to work properly :
- `ban`

## Example
```
$nomention
$unbanID
Successfully unbanned user!
```
