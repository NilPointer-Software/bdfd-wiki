# $suppressErrors
Blocks sending any error messages whenever an error occurs.

## Syntax
```
$suppressErrors
```

## Example
```
$nomention
$suppressErrors
$argsCheck[>1;Usage : ` !command <math expression> `]
$calculate[$message]
```
