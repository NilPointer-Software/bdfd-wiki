# $pinMessage
Pins the bot's response message in the current channel.

> ⚠️ A channel can have a maximum of 50 pinned messages.

## Syntax
```
$pinMessage
```

## Example
```
$nomention
$pinMessage
This is a cool pinned message! 😎
```
