# $max
Returns the largest number from the provided numbers.

## Usage
```
$max[A;B;...]
```

### Parameters 
- `A,B,...` `(Type: Integer || Flag: Required)`: The numbers to get the maximum from. Separate numbers using `;`.

## Example
```
$nomention
$max[100;20;50]
```
