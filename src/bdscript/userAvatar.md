# $userAvatar
Returns user's avatar URL.

## Syntax
```
$userAvatar[User ID]
```

### Parameters
- `User ID` : The user to return the avatar for.

## Example
```
$nomention
$image[$userAvatar[$mentioned[1;yes]]]
```
