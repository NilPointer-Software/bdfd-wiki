# $getTextSplitIndex 
Retrieves index from the provided value. Returns -1 if it couldn't find the value.

## Usage
``` 
$getTextSplitIndex[value]
``` 

### Breakdown 
- `value` - The value to search in the text split.

## Example 
``` 
$nomention 
$textSplit[Cake-Bread;-] 
$getTextSplitIndex[$message[1]] 
```
