# $modulo
Returns remainder between numbers.

## Syntax
```
$modulo[A;B]
```

### Parameters 
- `A` `(Type: Integer || Flag: Required)`: The dividend.
- `B` `(Type: Integer || Flag: Required)`: The divisor.
