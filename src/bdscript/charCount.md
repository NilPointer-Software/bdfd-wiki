# $charCount
Returns the amount of characters in the provided "Text".

## Syntax
```
$charCount[Text]
```

### Parameters
- `Text` `(Type: String || Flag: Emptiable)`: The text to return the character count for.

## Example
```
$nomention
Your message has **$charCount[$message]** characters.
```
