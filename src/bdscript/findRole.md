# $findRole
Finds a role's ID using the given role name, ID, or mention.

## Usage
```
$findRole[role]
```

### Parameters
- `role` `(Type: String || Flag: Emptiable)`: The role name, ID, or mention to find.

## Example
```
$nomention
$findRole[$message]
```
