# $displayName[]
Returns the display name of the given user.

## Syntax
```
$displayName[User ID]
```

### Parameters
- `User ID` `(Type: Snowflake || Flag: Required)`: The user to get the display name for.

## Example
```
$nomention
**$username[$message[1]]**'s display name is `$displayName[$message[1]]`
```
