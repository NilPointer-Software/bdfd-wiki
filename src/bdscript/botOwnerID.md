# $botOwnerID
Returns the bot owner's ID.

## Syntax
```
$botOwnerID
```

## Example
```
$nomention
My owner's ID: $botOwnerID
```

``` discord yaml
- user_id: 729343563401265193
  username: Nicky
  color: "#EE7908"
  content: |
    !botOwnerID
- user_id: 566613317972394004
  username: Wiki Bot
  color: "#748BD4"
  bot: true
  content: |
    My owner's ID: 729343563401265193
```
