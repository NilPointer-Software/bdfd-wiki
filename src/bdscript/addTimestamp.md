# $addTimestamp
Adds a timestamp to an embed.

## Usage
```
$addTimestamp[(index)]
```

### Breakdown
- `index` - What embed the timestamp should belong to. Default is `1`. [(learn more)](../resources/embedIndexes.md)

## Example
```
$description[Hi!]
$footer[That is the timestamp =>]
$addTimestamp
```
