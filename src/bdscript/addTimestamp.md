# $addTimestamp
Adds a timestamp to an embed.

## Syntax
```
$addTimestamp[(index)]
```

## Parameters
- `index`: To which embed the timestamp should be added to. _(Optional)_ [(learn more)](../resources/embedIndexes.md)

## Example
```
$nomention
$description[Hi!]
$footer[That is the timestamp =>]
$addTimestamp
```
