# $dmChannelID
Retrieves the DM channel ID for the provided user id.

## Syntax
```
$dmChannelID[user ID]
```
### Parameters
- `user ID` `(Type: Snowflake || Flag: Required)`: The user for whom to fetch the DM channel ID
