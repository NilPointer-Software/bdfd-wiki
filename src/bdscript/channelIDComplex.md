# $channelID[]
Returns channel ID for the given channel name.

## Syntax
```
$channelID[channel name]
```
### Parameters
- `channel name` `(Type: String || Flag: Required)`: The name of the channel.
## Example
```
$nomention
Channel ID: $channelID[$message]
```
