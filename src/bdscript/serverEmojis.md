# $serverEmojis
Returns all emojis of the provided guild.

## Syntax
```
$serverEmojis[Guild ID;Separator]
```

### Parameters
- `Guild ID`: The guild to return it's emojis.
- `Separator`: The separator to separate the emojis with.

## Example
```
$nomention
$serverEmojis[$guildID; ]
```
