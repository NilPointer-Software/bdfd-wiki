# $hypesquad
Returns an empty string or a lowercase name of the house the user is in.

## Usage
```
$hypesquad[User ID]
```
- `user id` - The user to get their hypesquad house.

### Example
```
$nomention
You are in $hypesquad[$authorID] house.
```
