# $hypesquad

Returns the hypesquad name of the provided user

## Usage

```

$hypesquad[userID]

```

### Breakdown

- `userID` - ID of the user whose name of the hypesquad should be returned

## Example

```

$hypesquad[$mentioned[1;yes]]

```
