# $hypesquad

Returns the hypesquad name of the provided user

## Usage

```

$hypesquad[userID]

```

### Parameters

- `userID` `(Type : Snowflake || Flag : Required)` : ID of the user whose hypesquad name should be returned

## Example

```
$nomention
You are in $hypesquad[$authorID] house.
```
