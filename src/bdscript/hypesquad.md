# $hypesquad
Returns the hypesquad house name of the provided user.

## Syntax
```
$hypesquad[User ID]
```

### Parameters
- `User ID` `(Type: Snowflake || Flag: Required)`: The ID of the user whose hypesquad house name should be returned.

## Example
```
$nomention
You are in $hypesquad[$authorID] house.
```
