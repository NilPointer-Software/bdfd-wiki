# $unescape
Unescapes semicolons `;` from the provided text. It allows interpreting of escaped semicolons as unescaped ones.

## Syntax
```
$unescape[Text]
```

### Parameters
- `Text` : The text to unescape.

## Example
```
$nomention
$randomText[$unescape[$message]]
```
