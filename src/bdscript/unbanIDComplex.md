# $unbanID[]
Unbans a user by using their ID.

## Syntax
```
$unbanID[User ID]
```

### Parameters
- `User ID` `(Type: Snowflake || Flag: Required)`: The ID of the user to unban.

### Permissions
Required permissions that the bot must have for this function to work properly :
- `ban`

## Example
```
$nomention
$unbanID[$message[1]]
Successfully unbanned user!
```
