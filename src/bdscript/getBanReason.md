# $getBanReason[]
Gets the user's ban reason.


## Usage
```
$getBanReason[User ID;(Guild ID)]
```

### Parameters
- `User ID`: User to get the ban reason for
- `Guild ID`: The server id from which to get the ban reason. Optional


#### Example
```
$nomention
Ban Reason: $getBanReason[154148273307910144]
```
