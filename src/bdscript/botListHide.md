# $botListHide
Hides this command from being shown on the BDL command list (if the bot is on [Bot Designer List](https://botdesignerlist.com)).

## Usage
```php
$botListHide
```

## Example
```
$nomention
This is a secret command! 🤫
$botListHide
```
