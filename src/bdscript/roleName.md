# $roleName
Returns a role's name using its ID.

## Syntax
```
$roleName[Role ID]
```

### Parameters
- `Role ID` `(Type: Snowflake || Flag: Required)`: The role for which to get its name.

## Example
```
$nomention
$roleName[$highestRole[$authorID]]
```
