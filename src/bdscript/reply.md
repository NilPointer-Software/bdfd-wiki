# $reply
Replies to the user's command

## Usage
```
$reply
```

## Usage #2
```
$reply[Channel ID;Message ID]
```
### Breakdown
- `Channel ID` - The channel where to find `Message ID` in.
- `Message ID` - The message to reply to.
