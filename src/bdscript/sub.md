# $sub
Returns the subtraction of the provided numbers.

## Syntax
```
$sub[Number;...]
```

### Parameters
- `Number` : The numbers to subtract. Separate multiple values using `;`.

## Example
```
$nomention
$sub[20;10;5]
```
