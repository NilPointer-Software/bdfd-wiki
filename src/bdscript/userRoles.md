# $userRoles
Returns all role names given to the user.

## Syntax
```
$userRoles[User ID]
```

### Parameters
- `User ID` : The user to return the roles for.

## Example
```
$nomention
$description[<@$mentioned[1;yes]>'s roles: 
$userRoles[$mentioned[1;yes]]]
```
