# $userRoles
Returns a user's roles.

## Usage
```
$userRoles[userID]
```

### Breakdown
- `userID` - The user to return the roles for.

## Example
```
$nomention
$description[<@$mentioned[1;yes]>'s roles: 
$userRoles[$mentioned[1;yes]]]
```
