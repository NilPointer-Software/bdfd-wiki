# $mentionedRoles
Returns the ID of the mentioned role.

## Syntax
```
$mentionedRoles[Mention number]
```

### Parameters 
- `Mention number` `(Type: HowMany || Flag: Required)`: The role-mention to get from the author's message (`1` = first role-mention, `2` = second, etc).

## Example
```
$nomention
$mentionedRoles[1]
```
