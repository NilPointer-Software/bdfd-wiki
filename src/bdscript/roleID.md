# $roleID
Returns a role's ID using its name.

## Usage
```
$roleID[roleName]
```

### Parameters
- `roleName` `(Type : String || Flag : Required)` : The role name to get the role ID for.

## Example
```
$nomention
Role ID For "$message": $roleID[$message]
```
