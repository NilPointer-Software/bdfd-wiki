# $roleID
Returns a role's ID using its name.

## Syntax
```
$roleID[Role name]
```

### Parameters
- `Role name` `(Type: String || Flag: Required)`: The name of the role for which to get its ID.

## Example
```
$nomention
Role ID For "$message": $roleID[$message]
```
