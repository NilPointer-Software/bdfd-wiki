# $or
Returns 'true' if at least one of the provided conditions is true, otherwise 'false' is returned.

## Usage
```
$or[condition1;condition2;etc]
```

## Example
```
$nomention
$if[$or[$message==hi;$message==hey;$message==hello]==true]
Hello $username!
$endif
```
