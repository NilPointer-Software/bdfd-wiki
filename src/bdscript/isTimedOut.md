# $isTimedOut
Returns "true" if the user is timed out, otherwise "false".

## Syntax
```
$isTimedOut[User ID]
```

### Parameters
- `User ID`: The user to check for.

## Example
```
$nomention
User is timed out: $isTimedOut[$mentioned[1]]
```
