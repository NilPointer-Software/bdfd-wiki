# $varExists
Checks if a variable exists. Returns `true` if it exists, otherwise `false`.

## Syntax
```
$varExists[Name]
```

### Parameters
- `Name` `(Type: String || Flag: Required)`: The variable's name to check.

## Example
```
$nomention
$varExists[$message]
```
