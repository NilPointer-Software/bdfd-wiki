# $calculate
Calculates a math expression.

## Syntax
```
$calculate[expression]
```

### Parameters
- `expression` `(Type: String || Flag: Required)`: The math expression to solve.

## Signs
- `+` - Addition.
- `-` - Subtraction.
- `/` - Division.
- `*` - Multiplication.
- `%` - Modulo.
- `**` - Power.
- `()` - Parentheses you can put equations in.
