# $username[]
Returns the username for the provided user ID.

## Syntax
```
$username[User ID]
```

### Parameters
- `User ID` `(Type: Snowflake || Flag: Required)`: The user to get the username for.

## Example
```
$nomention
$username just hugged $username[$mentioned[1]]!
```
