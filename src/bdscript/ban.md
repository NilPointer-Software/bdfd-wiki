# $ban
Bans the mentioned user.

## Usage
```
$ban[(optional) reason]
```

### Breakdown
- `reason` - The reason for this action, which will be sent to the audit-log. *(Optional)*

## Examples
#### Example #1
*(without optional fields)*
```
$nomention
$ban
<@$mentioned[1]> was banned!
```
