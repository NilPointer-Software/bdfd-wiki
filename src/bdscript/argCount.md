# $argCount
Returns how many words (aka arguments/args) are in the provided text.

## Syntax
```
$argCount[text]
```

 ### Parameters
- `text` `(Type: String || Flag: Emptiable)`: The text to get word count for.

## Example
```
$nomention
Word count: $argCount[$message]
```
