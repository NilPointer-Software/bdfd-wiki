# $serverIcon[]
Returns server icon for the given server ID.

## Syntax
```
$serverIcon[Server ID]
```

### Parameters 
- `Server ID` `(Type: Snowflake || Flag: Required)`: The server from which to get the icon.

## Example
```
$nomention
$image[$serverIcon[566363823137882154]]
```
