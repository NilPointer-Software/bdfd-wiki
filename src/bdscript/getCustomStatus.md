# $getCustomStatus
Returns a user's custom status.

## Syntax
```
$getCustomStatus[User ID]
```

### Parameters
- `User ID`: The user whose custom status to get.

## Example
```
$nomention
Your custom status: $getCustomStatus[$authorID]
```
