# $if
A block of code to be executed, if a specified condition is true.

## Syntax
```
$if[сondition]
```

### Parameters
- `сondition` `(Type: String || Flag: Required)`: please edit it

## Example
```
$nomention
$if[$authorID==$botOwnerID]
  You are the developer of this bot!
$endif
```
