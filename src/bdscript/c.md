# $c
Adds a comment to the code. Comments do not appear in the bot's response.

## Usage
```
$c[comment]
```
## Parameters 
- `comment` `(Type : String || Flag : Emptiable)` : add a comment to your code

 
## Example
```
$nomention
$noMentionMessage
$c[This is a say command. You are reading a comment!]
```
