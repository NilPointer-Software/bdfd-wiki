# $removeLinks[]
Removes all links from the provided text.

## Syntax
```
$removeLinks[text]
```

### Parameters
- `text` `(Type: String || Flag: Emptiable)`: The text from which all links will be removed.

## Example
```
$nomention
$removeLinks[$noMentionMessage]
```
