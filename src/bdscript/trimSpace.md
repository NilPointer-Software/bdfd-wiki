# $trimSpace
Removes all leading and trailing white-space characters from the provided text.

## Syntax
```
$trimSpace[Text]
```

### Parameters
- `Text` : The text which needs white-space characters removal.

## Example
```
$nomention
>$trimSpace[        Hi
        ]<
```
