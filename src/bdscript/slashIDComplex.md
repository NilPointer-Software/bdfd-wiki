# $slashID[]
Returns ID of the provided slash command.

## Syntax
```
$slashID[Slash command name]
```

### Parameters 
- `Slash command name` `(Type: String || Flag: Required)`: Name of the global slash command.

## Example
```
$nomention
ID of `$message` slash command: $slashID[$message]
```
