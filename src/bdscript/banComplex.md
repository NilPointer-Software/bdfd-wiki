# $ban
Bans the mentioned user with the reason.

## Syntax
```
$ban[reason]
```

### Parameters
- `reason`: The reason for this action, which will be sent to the audit-log.

## Example
```
$nomention
$ban[$noMentionMessage]
<@$mentioned[1]> was banned!
```
