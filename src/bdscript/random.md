# $random
Returns random numbers.

## Usages
There are two usages of the `$random` function.

### Usage #1
```
$random
```
Returns a random number between 0 and 9.

### Usage #2
```
$random[minimum;maximum]
````
Returns a random number between 'minimum' and 'maximum'.
