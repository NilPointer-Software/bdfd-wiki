# $removeLinks
Removes links from a value.

## Usages
There are two usages of the `$removeLinks` function.

### Usage #1
```
$removeLinks
```
Removes all links from the bot's reply.

### Usage #2
```
$removeLinks[text]
```
Removes all links from the provided 'text'.

## Example
```
$nomention
$removeLinks[$noMentionMessage]
```
