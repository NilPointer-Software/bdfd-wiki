# $checkCondition
Checks if a condition is `true` or `false`.

## Syntax
```
$checkCondition[Condition]
```

### Parameters 
- `Condition` `(Type: String || Flag: Required)`: The condition to check.

### Signs
`==` - Equal

`!=` - Not Equal

`<` -  Less Than

`>` - Greater Than

`>=` - Greater Than Or Equal To

`<=` - Less Than Or Equal To

## Example
```
$nomention
$checkCondition[$message==hello]
```
