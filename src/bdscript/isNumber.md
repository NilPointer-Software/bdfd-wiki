# $isNumber
Returns whether the provided value is a number or not.

> 🧙‍♂️ "true" means the value is a number, "false" means it isn't.

## Syntax
```
$isNumber[Value]
```

### Parameters
- `Value` `(Type: String || Flag: Emptiable)`: The text to check.

## Example
```
$nomention
$isNumber[$message]
```
