# $categoryCount[]
Returns the category count of a provided guild.

## Syntax
```
$categoryCount[guild ID]
```
## Parameters
- `guild ID`: The guild to get its category count.

## Example
```
$nomention
There are $categoryCount[$message[1]] categories in the server!
```
