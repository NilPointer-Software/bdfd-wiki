# $toLowercase
Converts the provided text to lowercase format.

## Syntax
```
$toLowercase[Text]
```

### Parameters
- `Text` : The text to convert.

## Example
```
$nomention
$toLowercase[THIS TEXT IS NOW lowercase]
```
